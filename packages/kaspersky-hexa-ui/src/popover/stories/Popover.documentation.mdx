В основе лежит Popover из Ant Design с темизацией Hexa UI. Компонент поддерживает позиционирование через **placement**, способ открытия **trigger** (`click`, `hover`, `focus`), ограничение размеров **shouldLimitSize**, кастомные **width** и **height**, а также настройку контейнера overlay через **getPopupContainer** для scrollable-областей. При открытии одного popover остальные автоматически закрываются."
