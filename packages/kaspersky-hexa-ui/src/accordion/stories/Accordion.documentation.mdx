Каждая секция — **AccordionPanel** с заголовком (**title**, **titleSize**) и опциональными **Tag**, **additionalText** и **actions** в шапке.

**withBorder** задаёт карточный вид панелей; без рамки — компактный список секций (**elementAfter** в макете только с рамкой). Если нужно переключение между равнозначными разделами без одновременного показа нескольких блоков, используйте **Tabs**. Для меню в заголовке — **actions** через **Dropdown**; клик по пункту не должен раскрывать панель (**stopPropagation** в **onClick**)."
