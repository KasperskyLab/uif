      title="Правила использования"
      description="**Select** применяется в формах и настройках, когда пользователю нужно выбрать одно или несколько значений из фиксированного набора вариантов, который слишком велик для радиокнопок или чекбоксов. Одиночный режим подходит для полей с однозначным выбором — страна, статус, категория. Множественный режим (`multiple`) используется, когда допускается выбор нескольких значений из предопределённых: теги, роли, разрешения. Режим `tags` подходит, когда пользователю нужна возможность вводить произвольные значения помимо готовых вариантов — например, email-адреса или ключевые слова.

Используйте **Field** или **FieldSet** в качестве обёртки, чтобы связать компонент с подписью. Если список опций подгружается динамически — используйте **loading** при ожидании данных и **loadingError** при ошибке. При проблемах с перекрытием выпадающего списка другими блоками или при скролле родительского контейнера — применяйте **usePortal** или **closeOnParentScroll**."
