Компонент остаётся в макете постоянно и не исчезает автоматически — используйте его для сообщений, которые должны оставаться видимыми до явного закрытия или изменения состояния.

Для краткой обратной связи после действий пользователя с автоматическим исчезновением используйте **Notification**. Для уведомлений на уровне всей страницы (ограничения лицензии, системные статусы) используйте **Alert**. **SectionMessage** лучше подходит для уведомлений внутри отдельных секций формы, карточки или панели."
