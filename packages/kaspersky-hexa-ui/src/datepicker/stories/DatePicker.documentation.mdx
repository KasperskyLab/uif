Для интервала «с — по» в макетах используйте **RangePicker**, а не два отдельных **Calendar**, если нужен единый контрол диапазона и пресеты периода. Для интервала только времени — **RangeTimeInput**. Для простого текстового ввода без календаря подойдёт **Textbox**; для выбора из фиксированного списка — **Select** или **SegmentedButton**."
