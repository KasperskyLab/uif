Выбранные значения — через **defaultValue** или **value** с **onChange**."

<Checkbox.Group direction="vertical" options={[
  { label: 'One', value: 1 },
  { label: 'Two', value: 2 }

<Checkbox.Group direction="horizontal" options={[
  { label: 'One', value: 1 },
  { label: 'Two', value: 2 }
]} />`}

<Checkbox.Group
  options={[
    { label: 'One', value: 1 },

  ]}
/>`}
    />

<Checkbox.Group
  defaultValue={[1, 2]}
  options={[
    { label: 'One', value: 1 },
    { label: 'Two', value: 2 },
    { label: 'Three', value: 3, description: 'Description in checkbox group' }
  ]}
/>`}
