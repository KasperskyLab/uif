      description="**AnchorNavigation** отображает длинный контент слева и фиксированную навигацию справа (256 px). Пункты задаются массивом **items**; каждый ***id*** должен совпадать с ***id*** соответствующего блока в **children**. При прокрутке активный пункт подсвечивается автоматически; клик или Enter по пункту плавно прокручивает к разделу. Панель навигации sticky и подстраивается под ближайший scrollable-родитель (например, **Sidebar**). Дополнительно настраиваются **title**, **className** и **style**."

    <DocExampleBlock

      description="Обязательный проп. Массив пунктов навигации. Каждый элемент содержит ***id*** (совпадает с ***id*** блока контента), **label** (подпись в навигации), опционально **children** (вложенные пункты) и **elementAfter** (узел справа от подписи). Длинные **label** обрезаются многоточием после двух строк (**TextReducer** с **lineClamp**=`2`). При пустом массиве панель навигации не рендерится."
      code={`
<AnchorNavigation
  items={[
    { id: 'overview', label: 'Overview' },
    { id: 'details', label: 'Details' },
    { id: 'settings', label: 'Settings' }
  ]}
>
<div style={{ width:"400px" }}>
  <Text id="overview" type="BTS4">Overview</Text>
  <Text id="details" type="BTS4">Details</Text>
  <Text id="settings" type="BTS4">Settings</Text>
</div>
</AnchorNavigation>`}
    />

    <DocExampleBlock
      title="children"
      description="Обязательный проп. Контент страницы слева от навигации. Блоки разделов должны иметь ***id***, совпадающие с ***id*** пунктов **items**, чтобы работали прокрутка и автоматическая подсветка."
      code={`
<AnchorNavigation items={[{ id: 'section-1', label: 'Section 1' }]}>
<div style={{ width:"400px" }}>
  <H6 id="section-1">Section 1</H6>
  <p>Section content</p>
</div>
</AnchorNavigation>`}

        { id: 'section-2-1', label: 'Section 2.1' },

      ]

>
<div style={{ width:"400px" }}>
  <H4 id="section-1">Section 1</H4>

  <H4 id="section-2">Section 2</H4>

</div>
</AnchorNavigation>`}
