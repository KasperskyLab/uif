Добро пожаловать! В данной папке хранятся исходники дизайн-системы [Hexa UI](https://kasperskylab.github.io/uif/hexa-ui/?path=/docs/intro--docs).

### Разработка
Для начала разработки компонентов достаточно локально поднять сторибук:

1. Установить зависимости (используется пакетный менеджер **npm**)
```
npm i
```
2. Запустить сторибук
```
npm run storybook
```

### Тестирование
Для запуска всех тестов можно воспользоваться командой
```
npm run test
```
Для запуска определенного теста воспользуйтесь командой
```
npm run test -t 'название теста в функции describe'
```

### Линтинг
Команда `npm run test` уже включает прогон линтера, отдельно его можно прогнать с помощью
```
npm run lint
```
Для автоматического исправления очевидных ошибок (например, форматирования, импортов и т.д.) можно воспользоваться командой
```
npm run lint-es:fix
```

### Деплой
Иногда возникает необходимость протестировать изменения в компонентах в продукте - для этого можно задеплоить локальную сборку в `node_modules` продукта. Процедура состоит из двух шагов

1. Создаем файл `.env`, где определяем переменную `deployPath` - путь до папки, куда будет скопирована локальная сборка пакета
```
deployPath={YOUR_PROJECT_PATH}/node_modules/@kaspersky/hexa-ui
```
2. Запускаем команду, которая соберет пакет и положит его в папку, указанную в `deployPath`
```
npm run release
```
