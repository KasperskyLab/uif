## О дизайн-системе

Hexa UI - дизайн-система, созданная с целью унификации визуального дизайна, проектирования и разработки пользовательских интерфейсов в Лаборатории Касперского.

Команда состоит из двух групп.

_Дизайн:_

Команда дизайн-системы B2B направления, являющаяся частью DRO. 

_Разработка:_

Команда разработки, являющаяся частью UIF - технологической платформы для построения пользовательских интерфейсов и веб-сервисов.
Подробнее можно прочитать [здесь](https://github.com/KasperskyLab/uif/blob/master/README.md).

Под капотом используются `styled-components` и `antd`.
