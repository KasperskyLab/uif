## 1.5.1
**Заменён импорт на прямой:**
``` js
// Было
import AccountIdentity from '@kaspersky/hexa-ui-icons/[esm, build/esm, ...]/24/AccountIdentity'
// Стало
import AccountIdentity from '@kaspersky/hexa-ui-icons/24/AccountIdentity'
```
**Не поддердивается импорт из индека:**
``` js
// Больше не поддерживается
import Icons from '@kaspersky/hexa-ui-icons'
import { Icons8 } from '@kaspersky/hexa-ui-icons'
// Доступен только импорт с указанием размера
import { AccountIdentity } from '@kaspersky/hexa-ui-icons/24'
import Icons24 from '@kaspersky/hexa-ui-icons/24'
```
